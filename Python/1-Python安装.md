# Python 安装
目前，Python 有两个版本，一个是 2.x 版，一个是 3.x 版，这两个版本是不兼容的。而 3.x 版越来越普及，所以教程将以最新的Python 3.x 版本为基础。

## Mac 安装
方法一：从Python官网下载安装程序 运行并安装；

方法二：如果安装了 Homebrew，直接通过命令 `brew install python3` 安装即可。
