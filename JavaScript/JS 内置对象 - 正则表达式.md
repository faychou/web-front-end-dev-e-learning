# 正则表达式

## 定义
``` js
//RegExp构造函数
const pattern=/\d/g;

//或 字面量形式
const pattern = new RegExp('\d','g')
```

### 修饰符
i

g

### 转义符号
某些符号代表一些特殊含义，如果要使用这些符号就必须要使用反斜杠来转义 `\` 这些字符。

## 分组和捕获
在正则中,圆括号有两种含义，一是用来分组，一是用来捕获想要的值。

### 分组
圆括号结合 * ？ + {} 使用时，是把圆括号内看作一个整体。

``` js
/(ab)+/  //匹配一个或多个ab

/(ab)+|(cd)+/   //匹配一个或多个 ab或cd
```
